Installation

1. Installez Python 3.8+ et exécutez :

pip install -r requirements.txt

Configuration

Vous pouvez configurer votre application en sélectionnant le service IA et la langue :

  • Langues disponibles : Français, Anglais, Espagnol, Arabe
  • Services IA : Ollama (local), OpenAI (ChatGPT), Claude (Anthropic)

Utilisation

Pour lancer l'application :

streamlit run app.py

Exemples

Voici un exemple de configuration d'un questionnaire :

{
    "title": "Satisfaction client 2024",
    "questions": [
        {"text": "Êtes-vous satisfait de notre service ?", "type": "choix unique", "options": ["Oui", "Non"]},
        {"text": "Quels aspects devons-nous améliorer ?", "type": "ouvert"}
    ]
}

Exportation

Vous pouvez exporter en JSON, CSV ou Excel :

survey.json, survey.csv, survey.xlsx

Dépannage

  • Erreur d'API : Vérifiez votre clé API
  • Application ne démarre pas : Installez les dépendances avec pip install -r requirements.txt
